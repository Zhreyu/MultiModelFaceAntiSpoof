### Installation
Setup
```
sudo apt install  git-lfs -y
git  clone https://github.com/sanjay7178/MultiModelFaceAntiSpoof
cd MultiModelFaceAntiSpoof
git-lfs pull
```
Inference
```
docker-compose  up  --build -d
```
